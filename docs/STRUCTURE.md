# Структура проекта

```
src/
├── api/            # Клиенты API и функции запросов
├── assets/         # Статические ресурсы (изображения, стили)
├── components/     # Vue компоненты
│   ├── layout/     # Компоненты разметки (Header, AuthLayout)
│   └── ui/         # Базовые UI компоненты (Button, Input, Card, Alert)
├── router/         # Конфигурация маршрутизации
├── stores/         # Pinia хранилища (Auth store)
├── types/          # TypeScript интерфейсы и типы
├── utils/          # Утилиты (Logger, helpers)
├── views/          # Страницы приложения (Login, Register, Profile)
├── App.vue         # Корневой компонент
└── main.ts         # Точка входа
```
